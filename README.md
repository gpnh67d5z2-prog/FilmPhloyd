# FilmPhloyd - Кинотека

Веб-сайт для поиска фильмов с использованием TMDB API.

## Возможности

- Поиск фильмов по названию
- Фильтрация по жанрам и годам
- Карусель популярных фильмов (авторотация каждые 5 секунд)
- Подробная информация о фильме (рейтинг, описание, бюджет, сборы)
- Пагинация результатов
- Адаптивный дизайн для всех устройств

## Технологии

- HTML5
- CSS3 (Grid, Flexbox, Media Queries)
- JavaScript ES6+ (Fetch API)
- TMDB API v3

## Структура

```
Film/
├── index.html
├── styles.css
├── script.js
└── README.md
```

## Использование

- Введите название фильма и нажмите "Поиск"
- Выберите жанр/год для фильтрации
- Кликните на фильм для просмотра деталей
- Используйте кнопки навигации для переключения страниц

## API

Использует TMDB API v3 для получения данных о фильмах

## Адаптивность

Оптимизировано для экранов: 1024px+, 768px, 600px, 480px, 380px
